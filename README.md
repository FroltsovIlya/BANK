_**Начало работы над веб.**_

Переход к Spring, появились HTTP запросы. Можно использовать Postman

Для запуска:
 1) скачать
 2) запустить докер энджайн
 3) сбилдить докер имейдж
 4) запустить контейнер
 5) перейти в постман или использовать curl для HTTP запросов.
    _Через localhost:8081/ping2 или localhost:8081/auth/ping можно проверить работоспособность веб-приложения быстро_
