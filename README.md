**_Черновая версия проекта Банка._**

Для запуска скачать папку src, перейти в ./src/app и запустить Main.java

В данной версии представлено:
    -ООП реализация архитектуры системы управления картами;
    -3 юзера: Админ и 2 пользователя;
    -Активация новых карт по запросу;
    -Блокировка карт по запросу;
    -Транзакции между картами;
    -Система авторизации пользователей;
    -Разделенные возможности для Админа и Юзера;
    -Разграничение прав для действий с картами по пользователям;
    -Определение истечения срока годности карт.

При запуске приложения Вас встречает диалоговое окно с просьбой ввода Юзернейма, затем необходимо ввести пароль. Текущие авторизационные данные:
Admin Admin
Ilya passwd
Vasya passwd3

У пользователя Ilya доступна 1 карта: 1000 2000 3000 4000, на которой доступно 10000 условных единиц денег.
У пользователя Vasya по умолчанию карт нет.
Admin - администратор системы.
